= Repor a palavra-passe do utilizador da aplicação
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Siga estes passos para redefinir a senha do usuário do aplicativo necessária para o registro do provedor SRA e VASA no vCenter Server usando as ONTAP tools for VMware vSphere.

.Passos
. Abra um navegador da web e acesse o seguinte endereço: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Faça login usando as credenciais de administrador configuradas durante a implantação das ferramentas ONTAP .
. Na barra lateral, selecione *Configurações*.
. Na página *Credenciais VASA/SRA*, selecione *Redefinir senha*.
. Digite e confirme a nova senha.
. Selecione *Redefinir* para aplicar a nova senha.
